一.机器学习的定义
Tom Mitchell：如果一个计算机程序在任务 T 上的性能 P 能通过经验 E 提高，则该程序从经验 E 中学习。

二.学习类型：
1.监督学习：训练数据包含输入和正确输出。典型任务包括回归与分类。
2.无监督学习：训练数据无标签，目标是发现数据结构，例如聚类。

三.线性回归
1.模型公式：预测值 = θ₀ + θ₁x
2.代价函数：J(θ) = (1/2m) ∑(h(x) - y)²
目标是最小化代价函数，找到最优参数 θ。

四.梯度下降
参数更新：

θ₀ ← θ₀ - α(1/m)∑(h(x) - y)
θ₁ ← θ₁ - α(1/m)∑(h(x) - y) * x

其中 α 为学习率。
